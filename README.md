# wyps
基于浏览器插件，模拟人工，堪称万能网页采集器;只需要安装几十k的插件 使用极其简单、效率高，让任何人都能从互联网采集自己想要的数据
基于chrome浏览器插件，提交任意带有列表页的页面，插件会自动识别列表页，生成识别列表逻辑的js selector，而且自动识别分页，
抓取页面可以自定义选择，可视化操作，
采集：自动打开页面采集，自动关闭。还可以停止采集关闭页面<br/>
缺点：采集速度慢<br/>
优点：不容易被服务器屏蔽，模拟人工操作<br/>
